# Prompt individual - Integrante 3 - MLOps / Experto

```text
Mi rol en el proyecto es Integrante 3 - MLOps / Experto.

Debes ayudarme con:
- Versionado de modelos.
- Arquitectura Champion/Challenger.
- Tests minimos.
- Docker.
- Red neuronal experimental.
- A/B Testing.
- Data Drift.
- Auto-reemplazo condicionado del modelo.

Prioridades:
1. No bloquear el Nivel Esencial.
2. Construir la capa experta de forma incremental.
3. Asegurar que el Champion estable nunca sea reemplazado sin cumplir reglas.
4. Documentar limitaciones tecnicas.

Reglas:
- La red neuronal es experimental: si no mejora, se documenta como comparacion.
- El drift no reemplaza modelos por si solo.
- El auto-reemplazo solo ocurre si el Challenger mejora metricas, mantiene overfitting < 5% y pasa tests.
- Docker y tests deben servir a la app real, no ser decorativos.

Cuando me ayudes, separa claramente:
- Que es obligatorio.
- Que es avanzado.
- Que es experto.
- Que puede quedar como simulacion academica documentada.

Despues de cada avance, indica:
- Archivos modificados.
- Comando de verificacion.
- Riesgo que reduce.
- Dependencias con ML Core o App / Producto.
```
