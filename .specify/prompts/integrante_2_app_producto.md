# Prompt individual - Integrante 2 - App / Producto

```text
Mi rol en el proyecto es Integrante 2 - App / Producto.

Debes ayudarme con:
- Diseno de la app.
- Formulario de inputs.
- Integracion del modelo Champion.
- Pantalla de resultado.
- Probabilidad/confianza de prediccion si aplica.
- Registro de feedback.
- Guardado de predicciones.
- Demo funcional.

Prioridades:
1. Crear primero una app minima funcional.
2. Asegurar que los inputs coincidan con el pipeline del modelo.
3. Mostrar resultado claro para usuario no tecnico.
4. Guardar feedback para futuros reentrenamientos.

Debo coordinar con ML Core:
- Lista definitiva de features.
- Tipos de datos.
- Valores permitidos.
- Modelo Champion y version.

Reglas:
- No inventar features.
- No modificar preprocessing critico sin validarlo con ML Core.
- No prometer resultados que el modelo aun no produce.
- Priorizar una app simple, estable y demostrable.

Cuando me ayudes:
- Indica que archivos de app se modifican.
- Explica como ejecutar la app.
- Incluye una prueba manual minima para verificar prediccion.
- Mantén el contrato de inputs alineado con .specify/2_spec.md.
```
